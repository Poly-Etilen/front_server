도서 소개

이 책은 최신 자바스크립트(ES2024) 문법부터 비동기 처리, 웹 컴포넌트, 프레임워크 없는 모던 프론트엔드 개발 방법론까지 모두 다룹니다. 자바스크립트의 깊은 원리를 이해하고 실제 서비스에 적용하는 방법을 배우고자 하는 모든 프론트엔드 개발자에게 필수적인 가이드입니다. 복잡한 내용을 시각적인 예시와 실습 코드로 쉽게 풀어내어 초급자와 숙련자 모두에게 유용합니다.


✨ 주요 내용:

  • 반복되는 자바스크립트 면접 질문 완벽 대비
  • Node.js를 활용한 백엔드와의 통합 방법
  • 성능 최적화 및 테스트 자동화 기법

목차

  1. Chapter 1. 자바스크립트의 기본 구조와 역사
  2. Chapter 2. 비동기 프로그래밍 (Promise, async/await)
  3. Chapter 3. 모던 웹 컴포넌트와 상태 관리
  4. Chapter 4. 테스트 자동화와 디버깅
  5. ...더보기
